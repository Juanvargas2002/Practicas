Lo aprendido de HTML

Etiquetas
  • head: informacion que interpreta el navegador, este contenido no lo ve el usuario directamente
  • body: Conteido que es mostrado al usuario
  • li: punto de lista
  • h*: titulo puede tener varios numeros desde el 1, mientras menor mayor sera el tamaño
  • aside: generalmente representa una sección de la pagina que no es el contenido principal, si no un acompañante
  • header: se usa para definir un encabezado de una sección generalmente
  • ul: contenedor de una lista sin orden
  • ol: contenedor de una lista ordenada
  • div: contendor basico generalmente usado como cajas con estilo flex o grid para organizar el interior
  • article: contenido que puede ser transportado a otra parte de la pagina y no perderia sentido su interior
